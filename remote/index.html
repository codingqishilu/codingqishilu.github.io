# 工具和服务

时间管理

  • Toggl - 免费的工作计时工具
  • Focus Timer - Mac 下一个简洁的番茄计时工具
  • 番茄土豆 - 结合了番茄（番茄工作法）与土豆（To-do List）的时间管理工具

项目协作

  • Trello - 轻量级团队流程协作和列表管理平台
  • Tower - 简单，好用的团队协作工具
  • Teambiton - 类 Trello 的简单高效的项目协作工具
  • Worktile - 让工作更简单
  • tita - 企业级的工作计划管理平台，其实我想说，它的首页真的好酷。
  • I Done This - 基于邮件的团队日报工具，小巧实用。
  • Coding.net - 一站式云端开发协作平台

沟通工具

  • Zoom - WebEx 创始人新创建的在线视频服务，多人视频很流畅
  • 小鱼易连 - 支持移动端、桌面端和微信小程序快速入会，提供高质量的视频会议服务。

工作场所

  • 墨社 - 即刻找到属于你的联合办公空间
  • WORKFUN - 边工作，边旅行

收款工具

  • BufPay.com 独立开发者收款平台 - 个人可用的即时到账收款接口
  • Xorpay.com 支付平台 - 个人签约支付宝、微信支付宝接口，安全稳定，合法合规
