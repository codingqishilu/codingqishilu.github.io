# 招聘要求

面向市场学习，补充自己的技能点。以下列出腾讯跟Bigo的招聘需求，若自己对某个公司有想法，可以针对性的写简历和准备相关的技能知识点

  • 腾讯

        计算机或相关专业本科及以上学历，3 年以上开发经验；
        基本功扎实，基础理论、算法、数据结构、网络协议（TCP/IP、HTTP），熟悉 OOP 编程模式，熟悉设计模式；
        精通 PHP 语言，熟悉 Laravel 框架；
        精通 Mysql、Memcache、Redis 等应用开发，精通 SQL 调优和数据结构设计；
        熟练使用 Linux 环境和 PHP 环境配置部署，熟悉 Shell 等脚本编写；
        熟练掌握 apache/nginx 等 web 服务器的配置、调优及问题定位分析；
        熟悉 web 安全相关内容并有相关实践经验；
        熟悉常用的系统可并发、高可用及性能优化的解决方案；
        熟悉 Reactjs，Vuejs 等框前端框架，前端工程化；
        熟悉使用 Docker、K8S；
        拥有良好的代码习惯，结构清晰，命名规范，逻辑性强，代码冗余率低；
        具有良好的对内、对外沟通协调能力，责任心强，踏实进取，追求卓越，具有良好分析问题解决问题能力
  • bigo

    岗位职责
    
    1、负责Web后端系统架构的建设及优化
    
    2、负责核心模块的架构设计及实现
    
    任职要求：
    
    1、本科及以上学历，计算机相关专业，拥有扎实的计算机基础理论知识； 
    
    2、具有PHP全栈开发设计经验，有良好的设计及编码能力；  
    
    3、掌握一些常见场景设计，如消息队列、缓存、高并发、高可用、高负载等；
    
    4、熟悉tcp/ip、http、websocket、dns等协议，能熟练使用抓包工具定位问题；
    
    5、熟练使用nginx、apache、haproxy、lvs等开源软件
    
    加分项
    
    1、掌握golang/java等其他一种后端开发语言
    
    2、熟悉JS，了解前端相关工具链
    
    3、深度使用过 swoole/openresty 等开源Web解决方案
    
    4、阅读过PHP源码，熟练开发PHP扩展
    
    5、有Web后端模块化、微服务化经验
